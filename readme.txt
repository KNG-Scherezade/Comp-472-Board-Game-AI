How to run on a windows 10 lab computer

1) open up command prompt(cmd)
2) move into directory Comp-472-Board-Game-AI-master (cd into the project root directory)
3) input and run: win10_venv\Scripts\activate.bat to put your computer into a python virtual machine
4) type: main.py
5) if it doesn't run, type: pip install numpy

this allows numpy to run on university computers
